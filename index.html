Configurar fuente de datos

Conecta una Google Sheet publicada como JSON para actualizar el dashboard automáticamente. La hoja debe tener la estructura de columnas: hospital, mes, indicador, numerador, denominador
1. En Google Sheets → Archivo → Compartir → Publicar en la web
2. Selecciona la hoja de datos → Formato CSV o JSON
3. Copia la URL y pégala arriba
4. Para formato JSON: reemplaza /pub? por /gviz/tq?tqx=out:json&
